# RealTimeX

RealTimeX is a real-time chat web application built using HTML, CSS, and Socket.IO.  
It enables instant message delivery through WebSockets with a clean and simple user interface.

## Tech Stack
- HTML
- CSS
- JavaScript
- Socket.IO
- Node.js

## Features
- Real-time messaging
- WebSocket-based communication
- Simple and responsive UI
- Scalable structure for future enhancements
